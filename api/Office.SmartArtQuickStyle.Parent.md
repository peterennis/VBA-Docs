# SmartArtQuickStyle.Parent property (Office)

Returns the calling object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ An expression that returns a **[SmartArtQuickStyle](Office.SmartArtQuickStyle.md)** object.
