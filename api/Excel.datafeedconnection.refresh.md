# DataFeedConnection.Refresh method (Excel)

Refreshes the data feed connection.


## Syntax

_expression_.**Refresh**

_expression_ A variable that represents a **[DataFeedConnection](Excel.datafeedconnection.md)** object.


## Return value

**VOID**
