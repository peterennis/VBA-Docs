# ThemeEffectScheme.Parent property (Office)

Gets the **Parent** object for the **ThemeEffectScheme** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ An expression that returns a **[ThemeEffectScheme](Office.ThemeEffectScheme.md)** object.


## Return value

Object
