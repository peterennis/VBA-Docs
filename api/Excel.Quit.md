# Quit method (Excel Graph)

Quits Graph.

## Syntax

_expression_.**Quit**

_expression_ Required. An expression that returns an **[Application](excel.application-graph-object.md)** object.


## Example

This example quits Graph.

```vb
myChart.Application.Quit
```
