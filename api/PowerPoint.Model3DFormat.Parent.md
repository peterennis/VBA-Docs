# Model3DFormat.Parent property (PowerPoint)

Returns the parent object for the specified object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ An expression that returns a **[Model3DFormat](PowerPoint.Model3DFormat.md)** object.


## Return value

Object
