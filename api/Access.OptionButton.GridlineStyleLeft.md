# OptionButton.GridlineStyleLeft property (Access)

Gets or sets the left gridline style of the specified option button. Read/write **Byte**.

## Syntax

_expression_.**GridlineStyleLeft**

_expression_ A variable that represents an **[OptionButton](Access.OptionButton.md)** object.


## Remarks

The **GridlineStyleLeft** property uses the following settings.

|Setting|Gridline type|
|:-----|:-----|
|0 (default)|Transparent|
|1|Solid|
|2|Dashes|
|3|Short dashes|
|4|Dots|
|5|Sparse dots|
|6|Dash dot|
|7|Dash dot dot|
