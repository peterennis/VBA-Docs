# ParagraphFormat2.Parent property (Office)

Gets the parent object for the **ParagraphFormat2** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ An expression that returns a **[ParagraphFormat2](Office.ParagraphFormat2.md)** object.


## Return value

Object
