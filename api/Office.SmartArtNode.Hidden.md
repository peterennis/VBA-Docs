# SmartArtNode.Hidden property (Office)

Returns **True** if this node is a hidden node in the data model. Read-only.


## Syntax

_expression_.**Hidden**

_expression_ An expression that returns a **[SmartArtNode](Office.SmartArtNode.md)** object.
