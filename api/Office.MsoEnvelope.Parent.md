# MsoEnvelope.Parent property (Office)

Gets the **Parent** object for the **MsoEnvelope** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents an **[MsoEnvelope](Office.MsoEnvelope.md)** object.


## Return value

Object
