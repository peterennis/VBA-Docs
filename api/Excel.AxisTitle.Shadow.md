# AxisTitle.Shadow property (Excel)

Returns or sets a **Boolean** value that determines if the object has a shadow.


## Syntax

_expression_.**Shadow**

_expression_ A variable that represents an **[AxisTitle](Excel.AxisTitle(object).md)** object.
