# TableObject.EnableRefresh property (Excel)

Specifies if the query table can be refreshed by the user. Read/write **Boolean**.


## Syntax

_expression_.**EnableRefresh**

_expression_ A variable that represents a **[TableObject](Excel.tableobject.md)** object.


## Property value

**BOOL**
