# MsoBlogImageType enumeration (Office)

Specifies blog image types.

<br/>

|Name|Value|Description|
|:-----|:-----|:-----|
|**msoBlogImageTypeGIF**|2|GIF image|
|**msoBlogImageTypeJPEG**|1|JPEG image|
|**msoBlogImageTypePNG**|3|PNG image|
