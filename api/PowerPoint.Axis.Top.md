# Axis.Top property (PowerPoint)

Returns the distance, in [points](../language/glossary/vbe-glossary.md#point), from the top edge of the object to the top of row 1 (on a worksheet) or the top of the chart area (on a chart). Read-only  **Double**.


## Syntax

_expression_.**Top**

_expression_ A variable that represents an '[Axis](PowerPoint.Axis.md)' object.
